Create or Updated a member in the configure Mailchimp list,

  • If the member DON'T EXISTS in the list it will be added to it.
  • If the member EXITS in the list and $update is TRUE. Member in the list will be updated with data in the DB
  • IF the member EXITS in the list and $update is FALSE. No data will be send to MailChimp

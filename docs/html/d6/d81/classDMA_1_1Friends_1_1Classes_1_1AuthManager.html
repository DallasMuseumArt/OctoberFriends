Authenticate a user by either username, email, or member id

Parameters
array$dataAn array of attributes to authenticate a user minimially requires the following
  • login Provide either the email address or username to authenticate. This setting is configured in the administrative backend or
  • member_id The member id stored in the usermeta->current_member_number field
  • password A password
  • no_password If true password authentication will be bypassed. Use with caution as this can be a security breach if used incorrectly.
array$rulesA set of validation rules to validate against see http://laravel.com/docs/5.1/validation
Returns
boolean returns true if the user is authenticated
